## 🔧 CUSTOMIZATION

### Change Language Strings:
```
1. Edit: js/i18n/en.json
2. Change English text
3. Translate to other languages
4. Refresh page ✅
```

### Add New Language:
```
1. Create: js/i18n/xx.json (xx = lang code)
2. Copy from en.json and translate
3. Add button to language dropdown
```
